# Assets Directory

This directory contains visual assets for CodyStack.

## Required Assets

- `banner.png` - GitHub README banner (1280x640)
- `icon.png` - App icon/favicon (512x512)
- `screenshot-dashboard.png` - Main dashboard screenshot
- `screenshot-marketplace.png` - Plugin marketplace screenshot
- `screenshot-workflow.png` - Workflow builder screenshot

## Design Guidelines

- **Color Palette**: Dark cyberpunk with electric blue (#00D4FF) and magenta (#FF3366) accents
- **Style**: GlassMorphic, frosted glass effects
- **Background**: Deep purple/black gradients
